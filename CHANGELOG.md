# Changelog

All notable changes to Gargadusa Companion are documented here.

## [1.0.0] – 2026-05-07

### Added
- Initial release of Gargadusa Companion
- Guild Synergy Optimizer with drag-and-drop party builder
- Save file editor with full adventurer/staff/gatherer/crafter support
- Dark and light theme support
- Native file import/export dialogs (main-process IPC)
- Cross-platform builds: Windows, macOS, Linux
- GitHub Actions CI/CD pipeline for automated releases
